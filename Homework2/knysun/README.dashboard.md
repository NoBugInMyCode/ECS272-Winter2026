## How to run (VanillaJS)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start dev server:
   ```bash
   npm run dev
   ```

3. Open the page:
   - http://localhost:3000

### Notes
- Data files are in `./data/*.csv` (copied from `data.zip`).
- This dashboard is static (HW2). For HW3 you can add UI widgets (dropdown/brush) and reuse these charts.
